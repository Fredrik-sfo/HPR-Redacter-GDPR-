# HPR-redacter

Lokalt verktyg for att maska tredjepartsuppgifter i HPR-filer (enskild firma), utan uppladdning.

## Anvandning
1. Oppna `HPR-redacter.html` i din webblasare.
2. Valj din HPR-fil.
3. Klicka "Analysera" och kontrollera forhandsgranskningen.
4. Skapa rensad fil och spara rapporten.

## Integritet
All bearbetning sker lokalt i webblasaren. Inga natverksanrop goras av sidan.
