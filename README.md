# HPR-redacter

Local tool for redacting third-party personal data in HPR files (sole proprietors), without uploads.

## Usage
1. Open `HPR-redacter.html` in your browser.
   If you want extra peace of mind, disconnect your internet connection while using the tool.
2. Select your HPR file.
3. Click "Analyze" and review the preview.
4. Create the redacted file and save the report.

## Privacy
All processing happens locally in your browser. The page makes no network requests.
